# Tawi Backend

## 1.Property Owners
```http
1. Get all Properties  - GET /api/users/get-properties
2. Get a property by ID - GET /api/users/get-property/{id}
3. Add a property - POST /api/users/add-property
4. Update a property - PUT /api/users/update-property/{id}
5. Delete a property - DELETE /api/users/delete-property/{id}
```
